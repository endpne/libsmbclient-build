# libsmbclient-build

build scripts for [libsmbclient](https://github.com/samba-team/samba)

## Installation

### Swift Package Manager

```
https://github.com/mpvkit/libsmbclient-build.git
```

## How to build

```bash
make build
# or build specified platforms 
make build platform=ios,macos
# or see help
make help
```
